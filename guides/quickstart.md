# Quickstart: Adopt TFWS v2 in 15 minutes

This guide shows the **minimal steps** required to publish Trust-First Web Standard (TFWS v2) signals on your domain.

No registration. No approval. No central authority.

---

## 1) Prepare your site

You need:
- A publicly reachable HTTPS website
- Ability to publish files under `/.well-known/`

This works with:
- Cloudflare Pages
- GitHub Pages
- Netlify
- Any standard web server (nginx, Apache, etc.)
